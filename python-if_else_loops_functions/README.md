# Python - If/Else, Loops, and Functions

## Description

This project introduces fundamental Python programming concepts such as conditional statements, loops, string manipulation, and basic functions. The exercises focus on understanding program flow, working with numbers and characters, and implementing simple logic using Python.

## Learning Objectives

By completing this project, you should be able to:

* Understand and use **if, elif, and else** conditional statements.
* Work with **loops** in Python.
* Manipulate **strings and characters**.
* Write and use **basic functions**.
* Use **ASCII values** with the `ord()` function.
* Implement logic without relying on built-in shortcuts like `str.upper()`.

## Project Structure

| File                        | Description                                                                |
| --------------------------- | -------------------------------------------------------------------------- |
| `0-positive_or_negative.py` | Prints whether a randomly generated number is positive, negative, or zero. |
| `1-last_digit.py`           | Prints the last digit of a random number and describes its value.          |
| `2-print_alphabet.py`       | Prints the lowercase alphabet using a loop.                                |
| `3-print_alphabt.py`        | Prints the lowercase alphabet except the letters **q** and **e**.          |
| `4-print_hexa.py`           | Prints numbers from 0 to 98 in both decimal and hexadecimal.               |
| `5-print_comb2.py`          | Prints numbers from **00 to 99** separated by commas.                      |
| `6-print_comb3.py`          | Prints all possible combinations of two different digits.                  |
| `7-islower.py`              | Function that checks if a character is lowercase.                          |
| `8-uppercase.py`            | Function that prints a string in uppercase without using `str.upper()`.    |
| `9-print_last_digit.py`     | Function that prints and returns the last digit of a number.               |
| `10-add.py`                 | Function that adds two integers and returns the result.                    |
| `11-pow.py`                 | Function that calculates a number raised to the power of another number.   |
| `12-fizzbuzz.py`            | Prints numbers from 1 to 100 following the **FizzBuzz** rules.             |

## Requirements

* Ubuntu 20.04 LTS
* Python 3.x
* All scripts must be executable.
* Code must follow **PEP8 style guidelines**.

## Running the Scripts

Make the files executable:

```
chmod +x *.py
```

Run a script:

```
./filename.py
```

Example:

```
./0-positive_or_negative.py
```
